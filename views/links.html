URL Formatting
  1. Any value from the hash can be included in the url. $item_name will be converted to Golden%20Frying%20Pan
  2. For query strings, ${item_name,quality} will be converted to ?item_name=Golden%20Frying%20Pan&quality=11
  3. If you need to change the key for a query string value, ${item_name:item,quality} will be converted to ?item=Golden%20Frying%20Pan&quality=11
  4. When using query strings, all boolean values will be converted to either a 1 or 0.
  5. For displaying links only when a specific attribute is available, such as for https://backpack.tf/effects/$particle_name, add a question mark after the dollar sign $?particle_name
